## 💡 About the Workshop

This hands-on intensive bootcamp was designed to bridge the gap between simple prompt engineering and scalable cloud deployment. During the session, we explored:
*   **Google AI Studio:** Fast-tracking multimodal prompt prototyping, context window optimization, and prompt-to-code exporting.
*   **Google Antigravity:** Building applications inside an AI-native agentic IDE where autonomous loops plan, execute, test, and self-heal software code bases.
*   **Cloud Agentic AI Tools:** Deploying these serverless agent pipelines directly onto Google Cloud infrastructure.

## 🤝 How to Contribute Your Learnings

We want this repository to live on as a community knowledge hub. Follow these simple steps to add your thoughts to the repository:

### 1. Fork and Clone the Repo
Fork this repository to your GitHub profile, then clone it locally:
```bash
git clone https://github.com
cd YOUR_REPO_NAME
```

### 2. Update `your_learnings.json`
Open the `your_learnings.json` file in your code editor and append your details to the JSON array using the following clean format:

```json
[
  {
    "name": "Your Name",
    "github_username": "your-github-handle",
    "biggest_takeaway": "What was the coolest thing you learned about Antigravity or AI Studio today?",
    "project_built_or_planned": "Brief description of the agent or web app you built during the hackathon phase."
  }
]
```

### 3. Commit and Submit a Pull Request
Push your changes back up to your forked repository and open a Pull Request (PR) against our `main` branch:
```bash
git add your_learnings.json
git commit -m "Add learnings from [Your Name]"
git push origin main
```

Once submitted, our mentors will review and merge your PR into the master community wall!
