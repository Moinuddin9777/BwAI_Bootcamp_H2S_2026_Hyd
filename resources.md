# 🛠️ Cloud Agentic AI, AI Studio & Antigravity: Resource Guide

Welcome to the **Build with AI Workshop** resource dashboard! This curated list contains official documentation, step-by-step code labs, and developer-focused video walkthroughs to take you from prototyping to fully orchestrating multi-agent systems.

---

## 📘 Official Documentation

### 🛠️ Google Antigravity & Agent Studio
* [Antigravity Core Documentation](https://antigravity.google)
  *This is the definitive engineering manual for Google's agentic IDE. It details how to write instruction sets (`AGENTS.md`, `SKILL.md`), manage background agent loops, and hand off tasks between different specialized AI workers.*
* [Antigravity Community Hub](https://antigravity.md)
  *An open-source developer hub containing community-contributed best practices for Antigravity. It includes extensive guides on safely setting up guardrails (`GUARDRAILS.md`) and letting agents interact with existing enterprise codebases.*
* [Vertex AI Agent Studio Guide](https://google.com)
  *Google Cloud’s corporate documentation for its enterprise, low-code agent creation platform. It teaches you how to visually design multi-agent workflows, connect them to external data sources, and export them as production-ready applications.*

### 💻 Google AI Studio & Gemini API
* [Google AI Studio Portal](https://google.dev)
  *The core web-based playground used by engineers to quickly prototype and experiment with Google's Gemini models. It allows you to refine system instructions, test complex multimodal prompts, and instantly grab raw API keys and code snippets.*
* [Gemini API Pathway Guide](https://google.com)
  *A curated learning path managed by Google developers to guide you through the foundational concepts of the Gemini API. You will learn the mechanics behind freeform inputs, structured JSON outputs, and persistent chat sessions.*

### 🧱 Agent Development Kit (ADK)
* [ADK Open Source Project](https://adk.dev)
  *The central repository and documentation home for the community-driven framework used to build production-ready AI agents. This resource provides the raw Python/TypeScript libraries needed to build custom agent tools and integrations.*
* [Enterprise ADK Framework documentation](https://google.com)
  *Google Cloud's developer documentation focusing on architectural patterns for large-scale agent orchestration. It provides blueprints on when to use predictable, graph-based agent structures versus dynamic, self-routing agent networks.*

## 🧪 Hands-On Codelabs

* 🚀 **[Build and Deploy to Google Cloud with Antigravity](https://google.com)**  
  *A hands-on, step-by-step tutorial that teaches you how to build a cloud-native, event-driven document processing system. You will learn to use Antigravity to automate connections between Google Cloud Storage, Cloud Run, Gemini, and BigQuery.*
* 🔄 **[AI Agent End-to-End SDLC Workshop](https://google.com)**  
  *An advanced engineering lab detailing the entire Software Development Life Cycle (SDLC) using AI agents. It guides you from the initial structural blueprint design down to building, fine-tuning, and evaluating custom ADK workflows.*
* ✍️ **[Text Prompting in Google AI Studio](https://google.com)**  
  *A highly practical introductory lab designed to master prompt engineering within the Google AI Studio interface. It teaches you how to translate vague, real-world business scenarios into strict, predictable system prompts and export them into active code.*
* 🌐 **[Launch Your Web App with AI](https://google.com)**  
  *A full-stack deployment lab that demonstrates how to assemble a standard web application architecture using AI tools. You will learn how to direct an agent to build a frontend, wire it up to a Firestore database, and containerize it for Cloud Run.*

## 📺 Videos & Technical Deep Dives

### 🤖 Antigravity Ecosystem
* [Google Antigravity 2.0: The Complete Developer Guide](https://youtube.com)  
  *A comprehensive presentation by Google's Vun Moan detailing the migration of Antigravity to a desktop-native environment. The session explains agent supervision models, how to configure background agent execution loops, and how to navigate the new command-line interface.*
* [Automate Your Entire SDLC with Google Antigravity](https://youtube.com)  
  *An official video showcase highlighting how autonomous agent networks handle complex engineering tasks. It walks through how multiple agents collaborate together to write clean code, handle debugging, and test full software feature updates entirely from scratch.*
* [Learn the Basics of Google Antigravity](https://youtube.com)  
  *A high-level introductory video designed for beginners entering the agentic AI landscape. It provides an immediate overview of how the Antigravity code editor interacts with agent orchestration and utilizes the environment's built-in sandbox web browser.*

### 🌩️ Cloud Architecture & ADK
* [Introducing Agent Development Kit (ADK)](https://youtube.com)  
  *A technical presentation introducing Google's open-source framework for building robust AI systems. This video focuses heavily on how to implement native multimodal streaming data and leverage open foundation agent templates hosted on GitHub.*
* [Building AI Agents at Scale on Google Cloud](https://youtube.com)  
  *A video masterclass hosted by the Google Cloud engineering team detailing production scaling patterns. It explicitly shows developers how to turn simple visual mockups built inside Agent Studio into robust, programmatic ADK production code.*
* [Building AI Agents on Google Cloud Run](https://youtube.com)  
  *A deep dive architectural video focusing on how to serve AI agents within serverless cloud infrastructure. It teaches key operational techniques such as managing short-term and long-term memory states, sandboxing code execution, and utilizing Cloud Run for scalable hosting.*
* [AI Agents Explained: Build Your First Agent in 8 Minutes](https://youtube.com)  
  *A concise, high-utility conceptual animation explaining the core logic loops of autonomous systems. It clearly illustrates how agent loops leverage automatic retries, tool execution, and code validation checkers to consistently produce flawless outputs.*
